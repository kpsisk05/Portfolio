    Background

    Terminal Trivia is a command line trivia game written in C++ as a personal project to strengthen my programming skills. It reads a custom formatted game file and runs entirely in the terminal, allowing multiple players or teams to compete in a text based environment. The project began as a way to practice object-oriented programming, file I/O, and modular design outside of coursework.

    Objective

    The goal of Terminal Trivia is to deliver an interactive multiplayer trivia game directly in the terminal. Players or teams take turns answering questions loaded from a specified game file, with the program tracking and displaying scores throughout the match. The design emphasizes portability, requiring only a C++ compiler and a text-based question file to run. Future plans include a Qt-based GUI to provide a more visual, user-friendly experience.

    Requirements

    • Runs from the command line using the syntax ./trivia <gameFile>.
    • Supports multiple players or teams with independent score tracking.
    • Parses a structured text file containing questions, multiple-choice answers, and correct responses.
    • Handles input validation and scoring for each round.
    • Compiles with a standard C++ compiler using the provided Makefile.

    Results

    Terminal Trivia successfully reads and parses game files, presents questions in a clear terminal interface, and maintains accurate player or team scores through an entire session. Its modular design separates logic into Question, Team, and Game classes making it easy to expand with new features or question sets. I am currently developing a graphical version using the Qt framework to expand the game beyond the terminal.

    Code

    The sorce code and a collection of game files can be found on my GitHub page found at Terminal Trivia

    How To Use

    1. Clone the git repository from my page at the link above
    2. Navigate to the repository in the terminal and run 'make'
    3. Run the game using ./trivia <gameFile> (game files can be found in the gameFiles folder.)
    4. Follow the prompts in the terminal and enjoy!
